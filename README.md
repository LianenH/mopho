#webmopho

就目前来说并没有达到计划目标
且因为时间匆忙，有使用ai生成代码片段
我一直很后悔我的高中年岁，我直到申请季的前两个月才开始做关于音乐技术的学习，我多么希望从高一时我就能有足够的认知来确定我的毕生目标。。我一直害怕我会带着一种悬而未定的模棱两可知道不存在的永远
我大概比拼不过那些大机构的末代精英们了，为什么申请本科的作品会比许多大学毕业生的作品要求都要高
